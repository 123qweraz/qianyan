拼音设置

输入与转换 候选显示模式 繁体输入功能 按键功能开关 双击辅码过滤 快捷键

输入与转换

分词分隔符
用于强制分割拼音音节的字符，如输入 "xi'an" 分为 "xi" 和 "an"
精准匹配自动上屏
当输入码唯一对应一个词时自动提交
自动造句起始音节数
拼音音节数达到多少个时开始自动组句（默认4，调大可减少误组句）
开启前缀联想
根据已输入的码推荐可能的后续词
开启简拼匹配
支持输入声母进行匹配（如 zm → 怎么）
纠错匹配
输入错误时自动纠错（如 zhonf → 中）
生僻字模式
控制是否显示生僻汉字（三级字库以外）
排序算法调试开关 — 关闭后对应因素不影响排序结果，仅用于排查问题
使用频率排序
关闭后按纯字典权重排序，不受历史输入影响
上下文联想排序
关闭后上文词不影响当前候选排序
模糊拼音
z/zh、c/ch、s/sh、an/ang 等模糊音匹配
模糊音设置 →
防呆模式
容忍常见拼写错误，智能模式自动纠错，严格模式要求精准输入
编辑器预览模式已移至 系统设置 → Linux 专属设置

候选显示模式

显示模式
选择候选词旁显示何种辅助信息
开启 繁体输出 后，汉字自动变为繁体，显示模式不变。
例如 "汉字 + 英语辅助码" 搭配繁体输出 → "繁體漢字 + 英語輔助碼"

繁体输入功能

繁体输出
开启后输出的中文为繁体字形
繁简体切换快捷键
按快捷键切换繁简体输出（需先开启繁体输出开关）。按住第一个键再按第二个键触发，如 CapsLock+F

按键功能开关

Shift 筛选模式
输入拼音时按 Shift 进入英文/全局筛选
键盘发音
打字时发出对应字母的声音
数字键选词
关闭后数字键将作为普通字符输入

双击辅码过滤

启用双击字母辅码过滤
快速双击任意字母键（如双击 h），等同于按 Shift + h，进入辅码过滤模式。候选列表只显示匹配该字母提示的候选字
双击间隔(毫秒)
两次按键的间隔低于此值判定为双击，默认 250ms

快捷键

长韵母快速输入 已移至独立页面 — 长韵母快速输入设置 →
翻到上一页
触发翻页的按键列表，使用逗号分隔
翻到下一页
触发翻页的按键列表，使用逗号分隔
选择上一词
触发选择上一个候选词的按键列表
选择下一词
触发选择下一个候选词的按键列表
以词定字（单键）
按指定键快速输入首个候选词中的第 N 个字，9=第1字，0=第2字
以词定字（Tab + 键）
按住 Tab 再按指定键快速输入首个候选词中的第 N 个字，1=第1字，2=第2字
Tab 导航模式快捷键（进入 Tab 导航模式后可用，详细说明见 导航模式帮助）
导航模式 - 删除
删除拼音末尾一个字符并重新查词（默认 D）
导航模式 - 模糊音切换
切换当前音节模糊音 zh↔z ch↔c sh↔s 并重新查词（默认 S）
导航模式 - 清空拼音
清空整个拼音缓冲区（默认 R）
启用 Ctrl + Space 切换输入法
经典的输入法切换组合键
启用 Ctrl + CapsLock 强制切英文并上屏
有拼音内容时直接上屏并切换至英文模式（和 Ctrl+Space 不同，会直接提交内容）
