键盘布局

⌨️ 说明
• 点击可视化键盘上的任意字母/符号键，编辑其按下、Shift、双击、长按行为。 • 点击「开始交换」后，再点击另一个键，双方按键行为整包互换。 • 「双击大写键列表」中配置的字母：双击该字母等同于 Shift+该字母，触发辅码过滤。 • 「长按大写键列表」同理：长按该字母等同于 Shift+该字母，触发辅码过滤。

⚙️ 按键行为设置

启用双击功能
开启后可识别快速双击按键并触发自定义行为
双击间隔(毫秒)
两次按键低于此间隔判定为双击，默认 250ms
双击大写键列表
双击这些字母键 = Shift + 该字母 → 触发辅码过滤。用逗号分隔，如 a,b,c
启用长按功能
开启后可识别长按按键并触发自定义行为
长按判定时间(毫秒)
按住超过此时间判定为长按，默认 400ms
长按大写键列表
长按这些字母键 = Shift + 该字母 → 触发辅码过滤。用逗号分隔
键盘发音
打字时发出对应字母的声音
