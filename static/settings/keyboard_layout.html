键盘布局

⌨️ 说明
• 点击可视化键盘上任一键，编辑其按下(tap)、Shift、双击、长按行为。
• 字母键的 tap 映射意味着该键始终输出映射文本（在空闲和拼音输入中均生效，nav 模式除外）。
• 「双击大写键列表」/「长按大写键列表」中的字母在自定义映射未设置时生效。

⚙️ 按键行为设置

启用双击功能
开启后可识别快速双击按键并触发自定义行为
双击间隔(毫秒)
两次按键低于此间隔判定为双击，默认 250ms
双击大写键列表
双击这些字母 = Shift+字母 → 辅码过滤。逗号分隔，如 a,b,c
启用长按功能
开启后可识别长按按键并触发自定义行为
长按判定时间(毫秒)
按住超过此时间判定为长按，默认 400ms
长按大写键列表
长按这些字母 = Shift+字母 → 辅码过滤。逗号分隔
键盘发音
打字时发出对应字母的声音
