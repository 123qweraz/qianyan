用户词典

💡 说明
• 用户词典 / 短期记忆 — 自动造词记录的自定义词组，最多保留 N 条，超出时自动淘汰使用次数最低的词
• 长期记忆 — 短期记忆中的词被使用达到 M 次后自动晋升，永久保留
• 上下文联想 — N-Gram 关联历史，记录前后词关系
使用下方标签页切换查看不同类型的数据。

⚙️ 排序与学习控制

用户选词顺序排序
选过的词自动排到该拼音候选的首位，关闭后完全按系统词典 weight 降序
MRU 记忆数量
最多记住最近选过的多少个词（默认2000）
上下文联想排序
根据上文词语关系调整候选顺序
启用造词记录
将上屏的多字词自动加入用户词典
学习阈值
同一个词出现多少次才自动收录（1=立即收录，默认2）
短期记忆容量
短期记忆最多保留的词条数（默认200）。超过时自动淘汰使用最少的词
晋升长期记忆次数
短期记忆中的词被使用达到此次数后晋升到长期记忆（默认5）
